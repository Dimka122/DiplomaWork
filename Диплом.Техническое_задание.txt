                             Техническое задание

1. Задание: 
     -Создать сайт по заказу суши,
  который позволит клиентам легко выбирать суши и оформлять заказы.
     -Сайт должен быть адаптирован для просмотра на различных устройствах,
  включая телефоны, планшеты и ПК. 

2. Название приложения:   DiplomaWork
     
    -https://github.com/Dimka122/DiplomaWork 

3. Основные требования приложения: 

  -Меню с разнообразием суши и информацией о каждой позиции.

  -Система регистрации и аутентификации пользователей.

  -Возможность добавлять выбранные позиции в корзину и оформлять заказ.

  -История заказов для зарегистрированных пользователей.

  -Административная панель для управления меню и заказами.

  -Оплата заказов.??????????

4.  Общие требования:

                          Меню
  -Просмотр списка доступных суши с информацией о каждой позиции 
  (название, описание, цена, изображение).
  -Возможность фильтровать суши по категориям.
  -Возможность просмотра дополнительных подробностей о позиции.

                   Регистрация и аутентификация
  -Регистрация новых пользователей с валидацией данных.
  -Вход в систему для зарегистрированных пользователей с использованием email и пароля.

                   Корзина и оформление заказа
  -Добавление выбранных суши в корзину.
  -Просмотр содержимого корзины с указанием общей суммы заказа.
  -Изменение количества и удаление позиций из корзины.
  -Оформление заказа с указанием адреса доставки или времени самовывоза.
  -Возможность применения скидок к заказу.

                       История заказов
  -Доступ к истории заказов для зарегистрированных пользователей.
  -Просмотр деталей предыдущих заказов.

                    Административная панель
 -Управление меню (добавление, удаление, редактирование позиций).
 -Просмотр и обработка заказов (изменение статуса, отмена заказа).
  
  5. Реализация приложения:

  -Язык программирования: C#
  -Фреймворк: ASP.NET Core
  -База данных: MS SQL Server
  -Адаптация под устройства: Bootstrap (или аналогичный CSS-фреймворк)


   -Клиентская часть (frontend) - интерфейс пользователя,
  реализованный с использованием HTML, CSS и JavaScript,
  а также адаптация под различные устройства.

   -Серверная часть (backend) - обработка запросов,
  взаимодействие с базой данных и бизнес-логика,
  реализованная на ASP.NET Core.

   -База данных - хранение информации о суши, пользователях и заказах.
